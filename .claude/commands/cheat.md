# Cheat Sheet

Tema específico (opcional): **$ARGUMENTS**

(Ejemplos:
- `/cheat` → mostrar el cheat-sheet completo
- `/cheat tasas` → solo la sección de tasas
- `/cheat Art 93` → solo lo relacionado al Art. 93
- `/cheat errores` → solo los errores típicos de parcial)

## Pasos a seguir

1. **Leer** `00-Sistema/Cheat-Sheet-Parcial.md`.

2. **Si NO hay argumentos**: mostrar el cheat-sheet completo respetando su formato (tablas, headers, callouts).

3. **Si HAY argumentos**: filtrar la sección o el contenido relacionado con esos términos. Mostrar solo lo relevante.

4. **Mantener el formato original**: tablas, listas, marcado con `==` y blockquotes.

5. **No agregar** información que no esté en el cheat-sheet. Si el usuario pregunta algo que no está, sugerir `/explicar [tema]` para buscar en las notas extendidas.
