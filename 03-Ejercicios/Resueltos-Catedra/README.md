## Cuándo agregar acá

Cuando un ejercicio resuelto por Claude (en `Resueltos-Claude/`) es chequeado y validado:
1. Movelo desde `Resueltos-Claude/` a esta carpeta.
2. Renombralo a `Material XX - Resuelto.md` (o agregalo al archivo existente si ya hay uno de ese material).
3. Borrá el original de `Resueltos-Claude/` para evitar duplicación.
