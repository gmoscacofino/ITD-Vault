# Resueltos - Claude

Carpeta donde **Claude guarda sus propias resoluciones** cuando se le pide resolver un ejercicio.

==**Estado inicial: vacía**==. Los archivos van apareciendo a medida que se piden resoluciones.

## Cómo funciona

1. **Pedís a Claude que resuelva un caso**:
   > Resolvé el Caso 1 del Material 9. Guardá la resolución en `03-Ejercicios/Resueltos-Claude/Material 09 - Caso I - Claude.md`.

2. **Claude genera el archivo** acá siguiendo el template de los Resueltos-Cátedra.

3. **Chequeás la resolución** con el profe / resolución oficial.

4. **Si está bien**: movés el archivo a `Resueltos-Catedra/` (renombrando si hace falta), y borrás el de acá.

5. **Si está mal**: o le pedís a Claude que corrija, o lo dejás como referencia de "qué no hacer" con una nota al inicio.

## Convención de nombres

`Material XX - Caso N - Claude.md` o `Material XX - Claude.md` si son varios casos juntos.

## Importante

==Todo lo que está en esta carpeta es PROPUESTA, NO VALIDADO==. No estudiar para parcial desde acá sin chequear.
