## Cómo usar este vault

### Si estudias por primera vez
1. Empieza por **[[Sistema Tributario Argentino]]** (panorama general).
2. Lee los **conceptos clave** transversales (Hecho Imponible, Manifestaciones, etc.).
3. Avanza por cada impuesto en orden: **Ganancias → IVA → IIBB → IDyCB**.

### Si estás repasando
- Salta directamente al impuesto/concepto donde tenés dudas.
- Usa los **`[[wikilinks]]`** dentro de cada nota para conectar conceptos.
- Para repaso rápido de antes del parcial: **[[Cheat-Sheet-Parcial]]**.

### Si estás preparando un caso
- Identificá los **4 elementos del Hecho Imponible** para cada impuesto en juego.
- Visualizá el árbol de decisión: **`Razonamiento-Por-Impuesto.excalidraw`** (requiere plugin Excalidraw en Obsidian).
- Buscá el impuesto específico en este índice.
