## Cómo está armado cada caso

Cada caso sigue el mismo template:

1. **Enunciado** — cita literal del PDF
2. **Conceptos involucrados** — `[[wikilinks]]` a las notas de teoría relacionadas
3. **Resolución** — paso a paso, con tablas y cálculos
4. **Respuesta final** — la conclusión clave resaltada
5. **Por qué este ejercicio** — qué evalúa, errores típicos, conexión con teoría

> El objetivo es que cada caso sirva tanto para **repasar el ejercicio** como para **entender el concepto que practica**.
